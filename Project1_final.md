# Project 1 – ANSUR II Analysis

## Author
Xiaojing Huang  

## Course
PBHLT 7101 – Data Management, Analytics and Visualization  

## Description
This repository contains the exploratory data analysis of the ANSUR II dataset conducted in Stata.

## Files Included
- Project1_final.do (Stata code)
- Project 1.docx (Final report)
- Project1_final.md (Markdown summary file)

## Overview of Analysis
The analysis includes data cleaning, recoding of missing values, BMI calculation, creation of BMI categories, derivation of body types, assessment of data quality, correlation analyses of anthropometric measures, and evaluation of reporting bias between self-reported and measured height and weight.
